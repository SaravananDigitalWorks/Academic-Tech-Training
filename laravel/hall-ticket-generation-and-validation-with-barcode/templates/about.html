About this Application

This application is developed using Python3.12. This application has two major modules. One for Student maintenance where administrator of this application can enter student details including student's image to maintain the student data. Other module will be used to validate the hall ticket by scanning the QR code available in the hall ticket.

Following technologies are used in this application.

  • Python Version 3.12
  • Python QR code library
  • Python's Flask Framework
  • HTML
  • CSS(Cascading Style Sheets)
  • Bootstrap
  • jQuery
  • Java Script
  • SQLite Database
  • MVC Architecture

This application has following two major modules

Student maintenace

Typical student details required for hall ticket such as name, register number and center details captured and stored in database. secret key for each student will be generated and stored along with student record. This secrect key will be embedded in QR code which is present in the hall ticket.

Hall Ticket Validation

Hall ticket can be scanned using google lense application. While scanning the QR code present in the hall ticket, url for hallticket validateion will be generated by google lens application. We can navigate this url by clicking on it will take through to the validation page. If the secret key is matching then the validation will be successful. We can get this result less than two seconds.
