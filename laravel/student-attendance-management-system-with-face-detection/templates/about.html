About this application

This application is developed using Python3.12 with two modules. One web based interface for maintaining employee details including their personal details with their face image. Other module which will scan each employee face and it will send signal to door opening module if the employee face is valid or security alert system module if the employee face is not valid.

Following technologies are used in this application.

  • Python Version 3.12
  • OpenCV Face detection library
  • Python's Flask Framework
  • HTML
  • CSS(Cascading Style Sheets)
  • Bootstrap
  • jQuery
  • Java Script
  • SQLite Database
  • MVC Architecture

Employee Maintenance

As part of employee onboarding process, administrator of this application, can add employee details using employee registration form. Using registration form, they can add user id, name, email, department, password and employee image. On submittion, these details are getting validated. Once all fields are validatied, employee information stored in the database.

Security scanning

This module is being attached to a live camera in front of the door. Each passing face is being captured by camera and passed to this module. This module compares incoming face images with existing models available in the system. If the incoming face image is matching with the model then it will send the success signal to door opening module. If there is no match then, alert signal will be sent to secutity module to alert unauthorized access.
