About this Application

This application is developed using Python3.12. This application has two major modules. One for menu maintenance where administrator of this application make entry of list of menus provided by the restaurent and its available timings. Other module for ordering menus from available menu list.

Following technologies are used in this application.

  • Python Version 3.12
  • Python's Flask Framework
  • HTML
  • CSS(Cascading Style Sheets)
  • Bootstrap
  • jQuery
  • Java Script
  • SQLite Database
  • MVC Architecture

Menu Maintenance

With this module, administrators can enter list of menus offered by the restaurant. Also the available timing of these menus will be selected for each menu. List, Add, Edit and Delete menu are the list of operations available in this module.

Order Maintenance

User can select the menus they want and they can update the quantity they want. Once they selected all menus they can submit for an order. They can see the total amout before submitting order. Upon submitting, an order will be generated with order id and unique reference number. User can view their order at any time by inputting the unique reference number.
