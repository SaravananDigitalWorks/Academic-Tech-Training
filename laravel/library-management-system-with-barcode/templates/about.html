About this application

This application is developed using Python3.12 with two modules. One for books maintenance with book details and another for book transaction maintenance like loan request, approve/reject loan request and receive books.

Following technologies are used in this application.

  • Python Version 3.12
  • Python barcode library
  • Python's Flask Framework
  • HTML
  • CSS(Cascading Style Sheets)
  • Bootstrap
  • jQuery
  • Java Script
  • SQLite Database
  • MVC Architecture

Book Maintenance

Once a new book is entered into library, administrator of this application, can make an entry in this system and add to the inventory. Once the book is entered into the inventory then it is available for users to search and loan the book

Loan books

With this module, user can request a book for loan if any copy of the book is available in the inventory. Administors will be able to review all loan requests and they can either approve or reject. They can also receive loaned books and mark it as received.
